# SIGIM - Sistema de Gestión de Informes Mensuales

Sistema de Gestión y Aprobación de Informes Mensuales para el Centro Agroempresarial y Desarrollo Pecuario del Huila (CAEDPH Yamboro) del SENA.

---

## 🔑 Credenciales de Prueba para Acceso

Puedes utilizar las siguientes credenciales por defecto para explorar el sistema:

### 👤 Perfil Instructor
*   **Correo Electrónico**: `instructor@sena.edu.co`
*   **Contraseña**: `1234`

### 👥 Perfil Coordinador
*   **Correo Electrónico**: `coordinador@sena.edu.co`
*   **Contraseña**: `1234`

> 💡 **Nota sobre el inicio de sesión**: 
> Adicionalmente, el sistema cuenta con un fallback dinámico que permite iniciar sesión como **Instructor** con cualquier correo que contenga la palabra `instructor` o `docente` (ej. `diego.instructor@sena.edu.co`), o como **Coordinador** con cualquier correo que contenga `coordinador` o `admin`, utilizando siempre la contraseña `1234`.

---

## 🚀 Tecnologías Principales
*   **React 19**
*   **Vite 8**
*   **Tailwind CSS v4** (con PostCSS y Autoprefixer)
*   **Lucide React** (Iconografía)
*   **Recharts** (Gráficos estadísticos de cumplimiento)
